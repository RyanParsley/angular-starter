# A sensible starting point for angular based single page apps

* Jade for markup
* Sass for styles
* Module based directory structure

## Serve it locally 

```
grunt watch:all
```

## Run tests

```
grunt watch:test
```

## Build a minified release

```
grunt release
```

## Install

```
npm install
```
